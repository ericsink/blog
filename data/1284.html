   CornSharp 0.5.1 (Sunday, 27 April 2003)  

CornSharp 0.5.1 is now available.  Changes since 0.5.0:

  • The Publish feature now brings up a dialog which shows all the files and which ones need to be published to the server.  You can change your mind and Cancel to abort the Publish operation.
  • bugfix:  There were some cases were smileys were not being converted to the smiley image.
  • The Edit Item dialog now has an HTML tab so you can see the markup.
  • Removed some extra whitespace being inserted by the WYSIWYG editor.
  • Added feature to allow the background color to be configured.
  • Added a LINK tag for RSS discovery.
  • Change the RSS link in the sidebar to use the little orange button.
  • Added sans-serif to the font face arial helvetica tags.
  • Fixed crash bug when you click OK in the appearance dialog with a background image configured.
