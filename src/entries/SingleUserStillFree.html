We get asked fairly often if we still offer free licenses for single users.  The answer is yes.

In fact, it's in our FAQ:

Do you still offer free licenses for single users?

Yes. Both Vault and Fortress are free of cost when used by a single user. Simply install the software and do not enter any serial numbers (license keys) at all. When no license keys are present, Vault and Fortress behave as if there is exactly one license available.

We first introduced this with the release of Vault 2.0 back in February 2004.

I should clarify here that this feature only works for a team of one.  For any team of N people where N>1, you will need to purchase N user licenses.

I should further clarify that when we refer to a "single user", we are not making reference to the issue of marital status.  :-)
