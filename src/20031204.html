We've got a Vault server here at SourceGear which we have setup so we can host projects for people in the community.  We call it VaultPub.  Right now there are only a couple of teams hosted there.  We'd like a few more.

If you have a community project and would be interested in moving your source control to VaultPub, first review the terms of use.  Among those terms, I'd like to highlight the two most important ones here:

  • When you use VaultPub, you are a beta tester of Vault.  The VaultPub server is currently running a pre-release version of Vault 2.0.

  • VaultPub cannot be used for private projects.  Anything you put on VaultPub will be visible to the entire world.

In addition to the terms of use, please note that VaultPub is not a full substitute for something like SourceForge or GotDotNet Workspaces.  VaultPub currently has source control only.  There is no facility for bug-tracking or forums or mailing lists or anything like that.  Yet.

If you are still interested, you can apply by simply sending me an email.  Please note also that we will probably not be able to accept everyone that applies.
