GUIs

There are many software developers who strongly prefer not to use a command-line interface. Most of them stopped reading this book several chapters ago.

Like it or not, the command-line nature of this book somewhat reflects current reality in the DVCS world. I’m not saying there are no graphical UIs for the popular DVCS tools. There are plenty of them. But I have the strong impression that distributed version control hasn’t got much traction yet among developers that don’t use the command line. This will change over time.
