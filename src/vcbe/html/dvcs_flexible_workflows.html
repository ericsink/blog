Flexible Workflows

Geography isn’t the only reason you might want to have more than one central server. You might also want to create a repository instance to support a specific purpose. Or you might want to use named branches to manage simultaneous work on more than one release. Bottom line: A DVCS offers incredible flexibility to support any kind of workflow your team needs. See Chapter 11, Workflows for more information.
